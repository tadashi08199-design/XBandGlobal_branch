# VISTAR Brand Guidelines

## 1. Core Identity
*   **Name:** VISTAR
*   **Tagline:** "VISTAR: Your star for global incorporation."
*   **Alternative Taglines:**
    *   "Navigate. Incorporate. Rise."
    *   "Verified global setup. VISTAR."

## 2. Meaning & Acronym
The name VISTAR stands for:
*   **V** = Verify / Validate / Venture / Vision
*   **I** = Incorporate / International / Identity / Idea
*   **S** = Setup / Start / Scale / Services
*   **T** = Trusted / Trade / Transform / Together
*   **A** = Abroad / Across / Anywhere / All
*   **R** = Register / Reliable / Reach / Ready

**Core Meaning:** "Verified Incorporation Setup, Trusted Across Regions"

## 3. Brand Rationale
*   **Visual Potential:** The name naturally evokes star and globe imagery, perfect for a global platform guiding businesses. Like a guiding star for international expansion.
*   **Premium Feel:** "Vista" (view) combined with "Star" creates a sophisticated, established presence suitable for a B2B legal/corporate services marketplace.
*   **Availability:** As a coined word, it offers a high probability of domain availability (e.g., `vistar.io`, `vistar.co`, `getvistar.com`) and low trademark conflict risk.

## 4. Design Direction (Initial Notes)
*   **Primary Colors:** Deep Blues (trust, corporate) and Gold/Yellow accents (the "Star").
*   **Typography:** Modern, clean, sans-serif suitable for SaaS (e.g., Inter, Plus Jakarta Sans).
*   **Logo Concept:** A stylized compass or star integrated with a globe or interconnected nodes representing cross-border verification.
