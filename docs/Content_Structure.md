# Content Structure for Countries

## Format (JSON)
Store this in the `countries` table `requirements` column.

## Example: United Arab Emirates (AE)
```json
{
  "company_types": ["LLC", "Free Zone", "Offshore"],
  "min_capital": "AED 0 (Varies by zone)",
  "local_director": "Required for Mainland (UAE National)",
  "registered_address": "Required (Provided by Free Zone)",
  "documents": ["Passport Copy", "Proof of Address", "Business Plan"],
  "timeline": "3-5 Working Days",
  "tax_info": "0% Personal Income, 9% Corporate (above threshold)"
}
