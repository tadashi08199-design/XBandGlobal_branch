# VISTAR - MVP

## Quick Start
1. **Clone:** `git clone [repo-url]`
2. **Install:** `npm install`
3. **Env:** Copy `.env.example` to `.env.local` and fill keys.
4. **DB:** Run `SCHEMA.sql` in Supabase SQL Editor.
5. **Run:** `npm run dev`

## Tech Stack
- Next.js 14, Supabase, Stripe, Tailwind.

## Deployment
- Push to `main` branch → Auto-deploy to Vercel.
- Ensure Stripe Webhook endpoint is set in Vercel env.
